# Proyecto-katas-python

## 1. 💻 Ejercicios de Python

En este proyecto se consideraron funciones en Python para el control de errores, manipulación de listas y creación de clases para representar objetos del mundo real. El problema general que se resuelve es la automatización de tareas repetitivas y la manipulación eficiente de información mediante programas claros, reutilizables y seguros.

### 2. 📖 Descripción del proyecto
Este proyecto consiste en resolución de ejercicios que abarcan el manejo de datos, procesamiento de textos, uso de funciones, control de errores, manipulación de listas y creación de clases para representar objetos del mundo real.

#### Objetivo del proyecto

El objetivo principal es desarrollar soluciones utilizando diferentes herramientas y conceptos fundamentales de Python con el fin de mejorar la capacidad de analizar un problema, diseñar una solución lógica y convertirla en un programa funcional.

#### Técnicas y enfoques utilizados

Para desarrollar las soluciones se utilizaron diferentes técnicas de programación:

#### Programación estructurada:
Uso de funciones, condicionales y bucles para organizar el código y resolver problemas paso a paso.

#### Funciones de orden superior:

Aplicación de:

* map() para transformar datos.
* filter() para seleccionar elementos que cumplen condiciones.
* reduce() para acumular resultados.

#### Funciones lambda:

Creación de funciones pequeñas y anónimas para realizar operaciones simples de forma más compacta.

#### Manejo de excepciones:

Uso de try, except y excepciones personalizadas para controlar errores como:
* Valores no numéricos.
* División entre cero.
* Listas vacías.
* Operaciones bancarias no permitidas.

#### Programación orientada a objetos (POO):

Creación de clases como:
* Arbol, para representar una estructura con tronco y ramas.
* UsuarioBanco, para simular operaciones financieras.

#### Manipulación de estructuras de datos:

Uso de:

* Listas para almacenar colecciones de datos.
* Diccionarios para representar información organizada.
* Tuplas para manejar conjuntos de valores.

En conjunto, el proyecto aplica diferentes herramientas de Python para resolver problemas prácticos mediante soluciones eficientes, reutilizables y organizadas.

### 5.  🏆 Resultados y Conclusiones

Durante el desarrollo del proyecto se implementaron múltiples ejercicios en Python enfocados en la manipulación de datos, automatización de tareas, validación de información y modelado de situaciones reales mediante programación estructurada y orientada a objetos.

1. Se desarrollaron funciones capaces de procesar listas, cadenas de texto y estructuras de datos sin intervención manual.

* Filtrado de estudiantes según calificaciones.
